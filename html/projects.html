GROUPOMANIA

Projet d'études

Projet final de la formation Développeur web dispensée par OpenClassrooms. Il s'agissait ici de créer une première version d'un réseau social d'entreprise.
Le but étant ici de mettre en place une application web complète en utilisant un framework adapté en front, communiquant lui avec un backend disposant d'une API Rest et d'une base de données MySql.

Projet d'études

Ma première rencontre avec JavaScript lors de ce projet d'études !
