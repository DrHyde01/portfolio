RESERVIA - REACT

Projet personnel

Transformer une simple intégration HTML / CSS en React ? Pourquoi pas !
A la base Reservia est un simple exercice de formation, demandant d'intégrer une maquette de manière responsive. Ici j'ai voulu recréer ce projet en employant mes nouvelles compétances.
Et quel gain de temps que d'utiliser un framework ou une librairie !

GROUPOMANIA

Projet d'études

Projet final de la formation Développeur web dispensée par OpenClassrooms. Il s'agissait ici de créer une première version d'un réseau social d'entreprise.
Le but étant ici de mettre en place une application web complète en utilisant un framework adapté en front, communiquant lui avec un backend disposant d'une API Rest et d'une base de données MySql.

ORINOCO

Projet d'études

Ma première rencontre avec JavaScript lors de ce projet d'études !
Celui-ci demandait de créer un site e-commerce à partir d'une API existante. Récupérer les articles de cette dernière, les afficher, mais aussi faire en sorte que l'utilisateur final puisse les mettre dans un panier et passer sa commande. Un projet enrichissant par bien des aspects.

OH MY FOOD !

Projet d'études

Second projet issu de ma formation, et sans doute le plus plaisant à mes yeux ! Il m'a permis de découvrir les fonctionnalités avancées de CSS, celles qui permettent d'avoir davantage de liberté dans la mise en page d'un site.
Grâce à l'utilisation de SASS et de la méthodologie BEM la structure du code a été rendue plus lisible, mieux organisée, rendant le projet plus maintenable !
